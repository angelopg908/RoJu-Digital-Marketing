# RoJu Studio

Landing page for RoJu Studio — a marketing agency for small and medium-sized businesses, based in San José, Costa Rica.

## About

Single-page site built with plain HTML, CSS, and a small amount of vanilla JavaScript. No build step, no frameworks.

## Sections

- Hero
- Services
- How we work
- Service details (Branding, Content creation, Social media)
- FAQ
- Final CTA
- Footer

## Stack

- HTML5
- CSS (custom properties, no preprocessor)
- Vanilla JavaScript (FAQ accordion, scroll reveal, mobile menu)
- Google Fonts: Fraunces, Inter, Caveat

## Files

- `index.html` — the full site (HTML, CSS, and JS in one file)
- `roju-logo.png` — brand logo (referenced in nav and footer)
- `newsletter-backup.html` — backup of the removed newsletter block, in case you want to put it back later

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
